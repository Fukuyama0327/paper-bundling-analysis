# Results Plan

このファイルは、査読論文・博士論文に向けて結果章で採用する図表と説明方針を整理するための作業メモである。

## 1. 結果章の中心

今回の論文化では、「広域連携・契約バンドリング効果の定量分析」を中心に置く。

結果章では、以下の順序で示すのが自然である。

1. 単一橋梁の劣化確率から期待契約件数関数を導く。
2. 閉形式の期待契約件数関数がシミュレーション結果を再現することを示す。
3. その関数を地域分割最適化に組み込み、距離制約と地域数の違いによる発注効率の変化を示す。
4. 現行管理単位と広域連携案を比較し、契約件数期待値の低減を定量化する。

## 2. Figure 2 候補: 期待契約件数関数

### 推奨

Word時点の分数関数フィッティング図をそのまま使うのではなく、閉形式の期待契約件数関数とシミュレーション結果の比較図へ更新する。

理由:
- フィッティング式より、閉形式の方が方法論として説明しやすい。
- \(L\) や \(q\) の意味を明示でき、契約バンドリングモデルとの接続が強い。
- 後続検討で作成済みの `simulation_vs_formula_comparison` があり、再現性を担保しやすい。

### 本文で示す式

```math
f(N,L)
=
\sum_{n=0}^{N}
\binom{N}{n} q^n (1-q)^{N-n}
\left\lceil \frac{n}{L} \right\rceil
```

閉形式比較で用いた値:

```text
L = 5
q = 0.012320362
```

代表値:

| N | f(N,5) |
|---:|---:|
| 1 | 0.012320 |
| 10 | 0.116593 |
| 25 | 0.266496 |
| 50 | 0.462007 |
| 100 | 0.712080 |
| 200 | 0.954904 |
| 323 | 1.194992 |

### シミュレーションとの比較

## 3. Figure 3 候補: 地域分割最適化結果

正本側の `図表作成/最適解v2.csv` から確認できた目的関数値を、現時点の結果候補とする。

| 条件 | 目的関数値 | 現行管理比の低減率 |
|---|---:|---:|
| 既存の管理 | 2.5241 | 0.0% |
| D=15km, M=6 | 2.0757 | 17.8% |
| D=20km, M=4 | 1.8086 | 28.3% |
| D=25km, M=3 | 1.5443 | 38.8% |
| D=30km, M=3 | 1.3848 | 45.1% |
| D=35km, M=3 | 1.2152 | 51.9% |
| D>=40km, M=1 | 1.1390 | 54.9% |

読み取り:
- 距離制約を緩和すると、目的関数値は一貫して低下する。
- \(D=25\) km, \(M=3\) でも現行管理比で約38.8%の低減が見込まれる。
- \(D \ge 40\) km では全橋梁を1地域に集約でき、目的関数値は最小となる。

注意:
- この表はWord正本側の結果であり、閉形式 \(f(N,L)\) に置き換えた再計算結果とは限らない。
- 査読論文化では、最終的に採用する目的関数とコード設定を一致させる。

## 4. 中間審査で追加された論点の扱い

今回の論文化では、中間審査で示した3研究のうち第1研究のみを対象にする。

本文に入れる:
- 発注効率
- 契約バンドリング
- 広域連携による契約件数期待値の低減
- 必要最小限の人員不足・担い手不足の背景説明

本文に入れない:
- 供給制約を明示的に組み込むモデル
- 人員不足を制約条件として扱う分析
- 新技術導入、予防保全、事後保全、ダウンサイジングなど第2・第3研究に属する議論
