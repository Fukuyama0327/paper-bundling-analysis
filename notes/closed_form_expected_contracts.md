## 閉形式の考え方

`compare_simulation_vs_formula.ipynb` では、マルコフ推移確率行列 \(P\) から、単一橋梁が1期間に補修を必要とする確率 \(q\) を求め、そのうえで地域内の補修発生橋梁数を二項分布として扱っている。

ノートブックの実行出力から、閉形式比較で使用した入力データは以下と確認した。

```text
N_values = 1, ..., 323
state_probs = [0.3250774, 0.6501548, 0.0247678]
L = 5
T = 100
S = 10000
seed = 123
```

推移確率行列:

```math
P =
\begin{pmatrix}
0.913206062 & 0.0861644998 & 0.000629438206 \\
0 & 0.985745578 & 0.0142544221 \\
0 & 0 & 1
\end{pmatrix}
```

この行列から得られる値:

```text
pi = [0.14194956, 0.85805044]
q = 0.012320
```

### 1. 単一橋梁の補修確率

推移確率行列 \(P\) を、補修不要状態間の推移行列 \(T\) と、補修状態への推移確率ベクトル \(r\) に分ける。

```math
P =
\begin{pmatrix}
T & r \\
0 & 1
\end{pmatrix}
```

ノートブックでは、補修不要状態の定常的な滞在分布 \(\pi\) を以下で計算している。

```math
\pi
=
\frac{
e_1^\top (I - T)^{-1}
}{
e_1^\top (I - T)^{-1}\mathbf{1}
}
```

そのうえで、単一橋梁が補修を必要とする確率を

```math
q = \pi r
```

としている。

### 2. 地域内補修本数の分布

地域内に \(N\) 橋梁があり、各橋梁が独立に確率 \(q\) で補修対象になると仮定すると、補修対象橋梁数 \(X\) は二項分布に従う。

```math
X \sim \mathrm{Binomial}(N, q)
```

したがって、

```math
\Pr(X=n)
=
\binom{N}{n} q^n (1-q)^{N-n}
```

である。

### 3. 契約バンドリング後の期待契約件数

1契約あたりの同時発注上限を \(L\) とすると、補修対象橋梁数が \(n\) 本のときの契約件数は

```math
C(n;L)
=
\left\lceil \frac{n}{L} \right\rceil
```

である。ただし \(n=0\) のときは \(C(0;L)=0\)。

期待契約件数は以下で計算される。

```math
f(N,L)
=
\sum_{n=0}^{N}
\binom{N}{n} q^n (1-q)^{N-n}
\left\lceil \frac{n}{L} \right\rceil
```

これが、後続更新で使っていた閉形式の期待契約件数関数である。

`output/comparison_summary.csv` によると、閉形式とシミュレーションの差は以下。

| l=3 | 3 | 323 | 0.009140 | 0.980903 | 0.009987 |

少なくとも \(L=5\) では、平均相対誤差が約0.75%であり、シミュレーション結果を閉形式でよく再現している。

## Word時点のフィッティング式との関係

Word・中間審査資料の

```math
f(N)
=
\frac{1.594N}{128.7+N}
```

は、シミュレーション結果に対する分数関数フィッティングである。

一方、後続更新で使うべき候補は、上記の二項分布に基づく

```math
f(N,L)
=
\sum_{n=0}^{N}
\binom{N}{n} q^n (1-q)^{N-n}
\left\lceil \frac{n}{L} \right\rceil
```

である。

論文化では、以下の構成が自然。

1. シミュレーションにより期待契約件数の挙動を確認した。
2. その後、同じ挙動を再現する閉形式の期待契約件数関数を導入した。
3. 最適化モデルには、フィッティング式ではなく閉形式 \(f(N,L)\) を組み込む。
4. シミュレーション結果との比較により、閉形式の近似妥当性を示す。
