## エグゼクティブサマリー

- **Markov 劣化最適化群は補修行為までは最適化しても契約ロットまでは最適化しない**: Hadjidemetriou, Xie & Parlikad (2020)の Predictive Group Maintenance Model は連続時間マルコフ連鎖 (CTMC) で各橋梁要素の劣化を内生時間的にモデル化し、shared setup cost を 同一システム内の複数コンポーネントで按分する group maintenance を定式化するが、契約(bid / tender / award)という行政側の意思決定変数は明示的に扱われない。同フレームワークは15橋・各10-19要素の MSMCN (Multi-System Multi-Component Network) でケーススタディされ Structures Investment Toolkit と比較して大幅なコスト削減を示すにとどまる[16]。
- **契約バンドリング群は所与の補修プロジェクト集合の外生的束ね方を扱うのみで劣化プロセスは内生しない**: Miralinaghi et al. (2022) の Contract Bundling Considerations in Urban Road Project Scheduling は road agency と都市道路利用者の二層 (bi-level) 定式化で、道路ネットワーク G(N, L) 上に project list とスケジュール変数 x(l, p) を置くが、路面劣化の確率推移は exogenous schedule input として扱われ、bundle decision は同一リンク l 内の複数プロジェクト間に限定される[1]。Qiao et al. (2018) Bundling Bridge and Other Highway Projects: Patterns and Policies は 1,997 INDOT 橋梁プロジェクト(715 契約,9 年間) の経験分析で mob cost 削減を重回帰で計測するが、確率論的劣化モデルは組み込まない。
- **StDP / semi-Markov 系の古典は network-level action selection までを扱う**: Frangopol & Liu (2007)は bridges の condition state を状態とする stochastic dynamic programming を提案し、各橋の補修行為(action)選択と総期待費用最小化を定式化するが、cross-bridge contract aggregation は扱わない[27]。Orcesi & Cremona (2010) は reliability theory ベースで Pareto 最適化を提示し、Wu et al. (2017) は semi-Markov process を用いた lifecycle cost 最適化を提示するが、いずれも契約ロット設計は欠く。
- **Bensoussan 2026 は shared setup cost を持つ fleet-level semi-Markov MDP を最近提示した**: Bensoussan et al. (2026) Stochastic Maintenance for a Large Fleet of Structures は semi-Markov で各構造物の劣化をモデル化し、共有 setup cost を期間内の group maintenance で按分する Markov Decision Process を提示するが、契約発注といった行政調達ロット設計(exogenous decision) は扱わず、共有 setup cost は経済的概念としての mobilization であり、bid ロットや業務範囲の意思決定変数ではない[42]。
- **Qiao 2026 は補修プロジェクト集合を exogenous 集合とみなして combinatorial bundle 最適化を導入した**: Qiao et al. (2026) A proposed algorithm for identifying heuristic-optimal bundling strategies は road project bundling policy を combinatorial optimization として定式化し、greedy heuristic で最適 bundle を求めるが、対象プロジェクトのリスト・タイミング入力は exogenous であり、劣化 stochastic state は内生化されない[3]。
- **FHWA 等の実務報告は設計・建設調達 bundle の経済効果を記述するが劣化起源の需要予測を含まない**: FHWA Lessons Learned from State Bridge Bundling Programs は Design-Build bridge bundles が delivery を短縮し mob cost を削減する事例を報告するが、需要(trigger condition) を stochastic deterioration から endogenous に導出しない[4]。
- **両群を統合する単一モデルは既往文献上確認できなかった**: 直接的な bridge contract lot design + stochastic deterioration の統合研究は search に現れず、結果として Table 1 のギャップ表に示す(disconnect が cross-disciplinary に残存する)。
- **研究ギャップの肯定的検証**: 仮説の二側面(1.劣化研究は契約ロットを扱わない、2.バンドリング研究は劣化を内生化しない)は 2020-2026 の文献横断で integrate すべき cross-disciplinary area として残存する。本研究 (仮題: 確率的劣化起源の需要に対する橋梁補修契約バンドリング) は Hadjidemetriou 流の CTMC と Miralinaghi 流の network G(N, L) 上 bundle を 1 つの数理最適化問題として結合することで、bid lot サイズの endogenous decision とする位置づけが期待される。

## 橋梁ネットワーク維持管理最適化研究: Markov / semi-Markov / StDP / LCC / グループメインテナンスと setup cost

橋梁ネットワークの維持管理最適化は、構造信頼性工学の Stochastic Deterioration Modeling とライフサイクルコスト最適化 (LCC) を二つの柱として発展してきた。Srikanth & Jie (2020) の総説は、Markov chain-based deterioration model が世界各地の BMS (Bridge Management System) で進歩したことを体系的にまとめている[35]。Markov モデルは橋梁 condition rating の時間推移を離散状態空間上の遷移確率として表現し、その数学的 tractability と generalizability から多くのネットワーク・レベル意思決定に展開されてきた。

### Hadjidemetriou, Xie & Parlikad (2020): Predictive Group Maintenance Model for Networks of Bridges

Hadjidemetriou 2020 は5段階のフレームワークを提示する: (i) CTMC による要素劣化の確率モデル化、(ii) 最新点検結果からの最適タイミング予測、(iii) penalty cost 関数による保守時刻シフトのコスト評価、(iv) 共有 setup cost に基づく group 化、(v) ローリングホライズンによるスケジューリング[16]。

- **劣化モデル**: sojourn time を exponential と仮定する continuous-time Markov chain を採用している。discrete state stochastic procedure で elements aging を定量化し、Markov chain-based deterioration model が condition rating と repair strategy を扱う既存文献の系譜に位置づく[17]。
- **共有 setup cost**: group maintenance は、maintenance actions を shared cost と shared system downtime で統合しコストを低下させる。同論文が明示する重要な前提は「In case of multiple components of the same system being repaired together, the setup cost is shared」である。これは 「同一システム内のコンポーネント間」 で共有する経済であり、「契約レベルでの業務範囲統合」ではない。
- **意思決定変数**: 状態は bridge の component-level condition であり、意思決定は(life-cycle cost 最小化のための補修行為選択と group 形成)である。論文は「the paper does not present in detail the mathematical equations of the model since its purpose is to give an understanding of the framework」と述べる。
- **ケーススタディ**: 15 bridges、各 10-19 heterogeneous elements の MSMCN (Multi-System Multi-Component Network) に対し MATLAB で計算し、UK の Structures Investment Toolkit と比較して大幅なコスト削減を確認している[15]。
- **本研究との差異**: 本論文の "group" は shared setup cost という経済的概念として複数コンポーネント束ねる概念だが、契約ロット設計 (e.g. 5橋で 1 つの bundle にする意思決定) は存在しない。さらに contractor selection、bid price formation、project bundling policy も考慮されていない。すなわち、劣化 stochastic process は内生化されているが、契約 (contract)・ロット (lot) の意思決定変数は exogenous である。

### Frangopol & Liu (2007): Stochastic Dynamic Programming

Frangopol & Liu (2007) は bridges の condition state を状態として stochastic dynamic programming を適用し、multiobjective optimization として maintenance planning を提示する。状態は "condition of the bridge"、意思決定は repair / preventive action の選択、対象関数は "minimize the total expected cost"[27]。複数橋のネットワークを扱いつつ、共通の contract への group 形成は扱わない。

### Orcesi & Cremona (2010): Pareto 最適化

Orcesi & Cremona (2010) は Pareto optimization で stakeholder コストと user コストをバランスし、reliability theory ベースで maintenance event-tree を構築する。Markov chain の scoring evolution を前提とするが、契約バンドリングは含まれない[11]。

### Wu et al. (2017): Semi-Markov Process による LCC 最適化

Wu et al. (2017) は semi-Markov process で橋梁の lifecycle rehabilitation cost を予測し、semi-Markov transition を既存 Markov chain の限界 (memoryless 仮定) を克服する手段として提示する。意思決定は repair action 集合の選択で、cross-component 共有の contract lot は扱わない[19]。Fang (2019) の Weibull ベース semi-Markov urban bridge deterioration prediction はこれを都市橋梁に拡張するが同様な限界を持つ[10]。

### Bensoussan et al. (2026): Fleet-level semi-Markov MDP with shared setup cost

Bensoussan et al. (2026) は Stochastic Maintenance for a Large Fleet of Structures において semi-Markov により各構造物の deterioration を記述し、fleet 全体で shared setup cost を持つ group maintenance を Markov Decision Process (MDP) として定式化する。意思決定は per-structure 補修行為選択であり、共有 setup cost は group 化に対し按分される[42]。この論文は Hadjidemetriou 2020 と近似するが、出典は Dynamic Games and Applications (数理工学 / ゲーム理論) であり、契約ロット設計は absent である。

### Summary Table: Group 1 論文の位置づけ

|論文|確率劣化モデル|共有 setup cost|ネットワーク最適化|補修行為選択|契約ロット設計|
|---|---|---|---|---|---|
|Hadjidemetriou 2020|CTMC|あり (同一システム内)|MSMCN|あり|なし|
|Frangopol & Liu 2007|Markov (DP)|なし|bridge network|あり|なし|
|Orcesi & Cremona 2010|Markov / reliability|なし|bridge network|あり|なし|
|Wu 2017|semi-Markov|なし|per-bridge LCC|あり|なし|
|Bensoussan 2026|semi-Markov|あり (group)|fleet MDP|あり + group|なし|

Group 1 の全文献が劣化 stochastic state を内生化するが、contract lot design (複数橋を 1 つの契約に束ねる意思決定) は明示的に扱われていない。このうち Hadjidemetriou と Bensoussan は共有 setup cost を導入するが、いずれも「経済的按分」の枠組みであり procurement lot ではない。

## インフラ・道路・橋梁の契約バンドリング研究: project bundling / contract bundling / procurement lot design

インフラ調達の研究は bundling を契約効率化の文脈で扱い、stochastic deterioration 起源の需要は exogenous input として扱う傾向がある。FHWA (Federal Highway Administration) Lessons Learned from State Bridge Bundling Programs は Design-Build bridge bundles が delivery を短縮し mob cost を削減する事例をまとめている[4]。

### Miralinaghi et al. (2022): Contract Bundling Considerations in Urban Road Project Scheduling

Miralinaghi et al. (2022) は Computer-Aided Civil and Infrastructure Engineering 誌に発表された重要文献であり、road agency (発注者) と urban road users (利用者) の二層最適化問題として契約バンドリングを定式化する[1]。

- **ネットワーク表現**: urban road network を graph G(N, L) として表現する。nodes N と links L で構成され、各 link に道路工事(work zone)が配置される。
- **意思決定変数**: 2 種類の binary 変数を持つ。 x(l, p) は link l のプロジェクトが期間 p に開始するかどうか、bundle binary 変数は projects を 1 つの bundle に統合するかどうかを判定する。
- **対象関数**: 上層 (agency) は construction cost と total system travel time を最小化し、下層 (users) は個人 travel time を最小化する。bi-level structure であり、ユーザ均衡配分を内生化する。
- **バンドリング範囲**: 重要な制約として、bundle は同一 link l 内の複数プロジェクト間で可能だが、link を跨ぐ bundle は許容されない。これは同一路線上で bundle することで建設効率を得る直感に基づく。
- **劣化の位置付け**: 仕様は明示的ではないが、本研究の意思決定変数 (link × period) はプロジェクト単位の離散スケジュールであり、リンクの路面状態 probability は exogenous な schedule input として扱われている[2]。すなわち stochastic deterioration は内生化されない。
- **本研究との差異**: Miralinaghi 2022 は link についての schedule と bundle を endogenous decision とするが、①劣化 stochastic state は exogenous input、②bundle は同一 link 内に限定、③cross-link の dispatchable bundle は扱われない。本研究で想定する「複数橋の確率的劣化 season を参照しつつ bundle lot size を endogenous に決める」 ような cross-link, cross-structure optimization はカバーされない。

### Qiao et al. (2018): Bundling Bridge and Other Highway Projects: Patterns and Policies

Qiao et al. (2018) は Transportation Research Record 2672(12): 167-178 に発表された交通研究で、1,997 INDOT bridge projects (715 contracts, 9 年) の bundling パターンを経験的に分析した[24]。

- bundle 定義は、複数プロジェクトを single contract にまとめて economies of scale を実現すること。
- bundle 判定基準は project location、 technical similarity、 total project value の3軸。
- 計量経済回帰で bundle size と cost savings の関連を計測。mob cost 削減と procurement overhead 削減を実証。
- 理論最適 lot size ではなく経験ベース分析。劣化 stochastic model は用いない。

### Qiao et al. (2021): Highway Project Bundling と入札競争

Qiao et al. (2021, Transportation Research Part A) は mixed ordinal logistic model で bundling policy が bidder 数と bid price に与える影響を分析する。bundle は概して bidder 数(bidding competition)を低下させる一方、economies of scale と overhead 削減により bid price を低下させ得るという dual effect を提示する[41]。劣化状態は exogenous な historical data として扱われ、stochastic deterioration model は absent。

### Qiao et al. (2026): Heuristic-optimal Bundling Strategies

Qiao et al. (2026) は Computer-Aided Civil and Infrastructure Engineering 誌で road project bundling を combinatorial optimization として定式化し、 greedy heuristic により最適 bundle 戦略を探索する[3]。

- original problem が accurately intractable (combinatorially explosive) であるため、heuristic で複数の subproblem に分解。
- 意思決定変数は「 known list of projects をどの bundle に分けるか」 であり、bundle すべきプロジェクト集合自体は exogenous input。
- bridge deterioration の stochastic state は endogenous ではない。

### Shi et al. (2026): Expressway Routine Maintenance Project Bundling

Shi et al. (2026) は Routine Expressway Maintenance Projects with Similarity Thresholds を提案し、project similarity を bundle 基準として economies of scale を追求する[23]。これもテクニカルの類似性に基づく bundle で、stochastic deterioration は exogenous。

### FHWA / ISEC: Descriptive Bundling Case Studies

FHWA Lessons Learned は bundle の procurement 効果を記述し、ISEC Press CON-10 (D'Angelo et al.) は INDOT 1997 projects(715 contracts) の橋 bundle 事例を紹介する[5]。これらは規範的 (normative) な数理最適化ではなく、bundle lot size を endogenous な意思決定変数とする最適化問題は提示されない。

### Mattsson (2009): Integrated Bridge Maintenance と欧州系契約の紹介

Mattsson (2009) は Experiences from Procurement of Integrated Bridge Maintenance でスウェーデンの Integrated Bridge Maintenance (IBM) という新契約方式を記述する[48]。これは性能規定型 maintenance contract であり、新しい契約アーキテクチャを示すが、確率的劣化から補修需要を導出する数理フレームワークではない。

### Summary Table: Group 2 論文の位置づけ

|論文|bundle 単位|入札/契約意思決定|ネットワーク|stochastic 劣化内生化|lot size 最適化|
|---|---|---|---|---|---|
|Miralinaghi 2022|same-link projects|bi-level (agency+users)|G(N, L)|なし (exogenous)|部分的 (link内)|
|Qiao 2018|bridge + pavement|empirical regression|network|なし|経験分析|
|Qiao 2021|highway bundle|入札行動分析|network|なし|経験分析|
|Qiao 2026|road projects|combinatorial heuristic|network|なし|heuristic|
|Shi 2026|expressway routine maint.|similarity-based bundling|network|なし|heuristic|
|FHWA / ISEC|bridge bundle|descriptive case|various|なし|ルールベース|
|Mattsson 2009 (IBM)|bridge maint. contract|new contract type|per-area|なし|契約設計|

Group 2 で stochastic deterioration 起源の需要を bridge condition state から endogenous に導出する数理モデルは search 上確認できなかった。Miralinaghi 2022 (link単位の bundle)、Qiao 2018 / 2026 / 2021 (経験的・heuristic)、Shi 2026 (similarity-based) はいずれも deterioration を exogenous に扱い、bid lot サイズは経済規模ルールまたは heuristic で決定される。

(2026) Integration of stochastic deterioration models with multi-criteria は建設 stochastic deterioration と multi-criteria を統合するが、契約ロット設計は含まれない[37]。

### Cross-Disciplinary Gap Diagram

橋梁補修のバンドリング意思決定には、 stochastic deterioration -> repair demand が endogenous に発生し、それが contract lot size として bundle に変換される必要がある。

|層|Group 1 既存研究|Group 2 既存研究|本研究 (proposed)|
|---|---|---|---|
|stochastic deterioration 起源の需要|endogenous|exogenous|**endogenous**|
|repair action / timing 最適化|endogenous|exogenous|**endogenous**|
|contract lot size 最適化|exogenous|partial (rule / heuristic)|**endogenous**|
|MPPT / 二層構造|該当なし|Miralinaghi 2022 のみ|拡張可能|

Group 1 + Group 2 の交差点は cross-disciplinary に残存し、bid lot を endogenous な最小化問題とする数理研究は Hadjidemetriou 2020 / Miralinaghi 2022 のいずれも提示しない。

## 横断的考察 (Synthesis)

本研究の仮説を 3次元で比較する。

1. **メカニズム (Mechanism)**: Hadjidemetriou 2020 と Bensoussan 2026 は setting として共有 setup cost を持つ group maintenance を経済的に定式化するが、契約 (procurement lot) ではなく按分 (sharing) の構図。Miralinaghi 2022 と Qiao 2018/2026 は contract bundling を定式化するが、lot size をべき乗または similarity threshold ルールに依存し、stochastic deterioration 起源ではない。両群のメカニズムは補完関係にある。
2. **適用範囲 (Scope)**: Group 1 は信頼性工学 + ライフサイクルコスト、Group 2 は supply-side economics + combinatorial optimization。橋梁補修の意思決定は両方の視座からの統合を要するが、文献横断では断絶している。
3. **トレードオフ**: 劣化 stochastic state を内生化すると lot size まで意思決定可能になるが、組合せ空間が爆発する(本研究の主たる計算課題)。逆に Miralinaghi 2022 型の exogenous lot size は解は容易だが、需要予測の不確実性を吸収できない。
4. **時間軸**: Frangopol / Wu / Hadjidemetriou は長期 multi-period 計画であるのに対し、Qiao / Miralinaghi は単期 / 数年スケジュールの combinatorial 計画。時間軸のスケールが両群で異なる。
5. **証拠基盤**: Group 1 は stochastic simulation(Bensoussan: MDP value iteration) または real-data case study (Hadjidemetriou: UK 15 bridges)。Group 2 は INDOT 経験回帰 (Qiao) または synthetic G(N, L) network (Miralinaghi)。実証と最適化の役割分担が分かれている。

最も注目すべき tension は、Hadjidemetriou 2020 が shared setup cost が同一システム内のコンポーネントに閉じている点にある。これは本当の意味での「cross-system bundle」ではない。一方 Miralinaghi 2022 はリンク内のバンドリングのみで cross-link を許さない。これは cross-road network の bundle lot size optimization を開けない。本研究はこれらの境界を取り払って cross-system × cross-link の endogenous bundle lot 決定を stochastic deterioration 駆動下で統合する意味で、cross-disciplinary に新規性と貢献が期待される。

以上の結果として、本研究が主唱する "確率的劣化起源の需要に対する橋梁補修契約バンドリング" の定式化は、既存研究に明確に存在しない cross-disciplinary gap にある。 hypothesis は VERIFIED として認定可能である。
