# 橋梁補修工事の契約バンドリング
## ——調達論点の体系的整理——

---

## 0. レポートの目的

橋梁の広域連携バンドリング最適化に関する査読論文執筆にあたり、以下の問いに答えるための調達論点に対して、実情を整理する。

- 発注効率化を期待契約件数で代理することは妥当か？
- 橋梁補修工事を対象業務とする理由は何か？
- 距離を目的関数ではなく制約条件に置く理由は何か？
- 契約バンドリングのメリット・デメリットはどう整理できるか？
- 発注者側効率化と受注者側効率化はどう区別するか？
- 地域数 \(M\)・供給制約はどう扱うか？

## 1. 橋梁維持管理業務の全体像と補修工事の位置付け

### 1-1. 発注行為の分類と発生構造

橋梁維持管理に関わる発注行為（発注者が契約単位として処理する業務）は以下の通りである。

| 発注行為 | 発生構造 | 1件あたりの規模感 | 発注頻度の性質 |
|---|---|---|---|
| 定期点検 | 道路法施行規則に基づき5年に1回が基本 | 橋梁単位〜複数橋梁まとめ | **確定的・周期的**（行政判断の余地小） |
| 日常点検・巡回 | 道路維持管理の一環として継続的に実施 | 路線単位・区域単位 | **継続的**（年次委託・包括委託が多い） |
| 詳細点検（近接目視等） | 定期点検で問題が発見された場合等 | 橋梁単位 | **条件付き発生**（定期点検の後工程） |
| 長寿命化修繕計画の策定・改定 | 5年サイクルを目安に改定 | 自治体単位・橋梁群単位 | **周期的**（計画業務、コンサル委託） |
| 補修設計 | 補修必要と診断された橋梁ごとに発生 | 橋梁単位〜複数橋梁まとめ | **確率的発生**（劣化状態に依存） |
| **補修工事** | **劣化に応じて確率的に発生** | **橋梁単位〜複数橋梁まとめ** | **確率的発生**（本研究の主対象） |
| 補強工事（耐震補強等） | 政策的判断・耐震診断結果に基づく | 橋梁単位 | **政策的発生**（予算・計画に依存） |
| 撤去工事 | 管理方針の判断（廃橋等）に基づく | 橋梁単位 | **個別判断型**（代替ルート確保等が必要） |
| 更新工事（架け替え等） | 長寿命化修繕計画に基づく大規模修繕 | 橋梁単位 | **計画的発生**（大規模・長工期） |

出典：[群マネ手引き](https://www.mlit.go.jp/report/press/content/001964898.pdf)、[包括委託手引き（2023）](https://www.mlit.go.jp/sogoseisaku/maintenance/_pdf/houkatsu_tebiki.pdf)、[土木学会ガイドライン（案）本編（2015）](https://committees.jsce.or.jp/cmc/node/107)

### 1-2. 補修工事を研究対象とする理由

1-1の分類を踏まえ、補修工事を本研究の対象とする理由を以下に整理する。

**① 確率的な発生型需要であり、制度的定期業務と性質が根本的に異なる**

橋梁の劣化は確率過程として記述でき、どの橋梁がいつ補修を必要とするかは事前に確定していない。これは定期点検（道路法施行規則に基づき5年に1回）のような制度的・周期的業務とは発生構造が異なる。補修需要の不確実性こそが、本研究が確率的モデルを用いる根拠であり、モデルと対象業務の整合性がある。

**② 小規模・点在的な発注が積み重なり、束ねる余地が大きい**

維持修繕工事の現場条件として「施工箇所点在」63.3%、「小規模工事」33.8%（[国総研](https://www.nilim.go.jp/lab/bcg/siryou/kpr/prn0063.htm)、p.30）と報告されており、橋梁工事はその典型例である。1-1で示した通り、補修工事は「仕様確定困難・設計変更頻発・入札参加者不足・監督難度高」という四重の困難を抱える発注行為であり、個別橋梁単位の小規模発注が続くと調達管理負担が累積する。複数橋まとめての発注は、こうした構造的課題に対する実務的な応答として既に模索されており（[四国地方整備局, 2019](https://www.skr.mlit.go.jp/etc/R0108_kyoryohosyu.pdf)）、本研究の集約最適化にはその余地がある。

**③ 国内外に実務事例・先行研究の蓄積がある**

米国では複数橋梁の一括発注（bridge bundling）を実施している州が多く、架替・補修・長寿命化工事（rehabilitation）を含む実務事例が蓄積されている（[FHWA Factsheet](https://www.fhwa.dot.gov/innovation/everydaycounts/edc_5/docs/project-bundling-factsheet.pdf)、[FHWA Lessons Learned](https://www.fhwa.dot.gov/ipd/alternative_project_delivery/defined/bundled_facilities/lessons_learned.aspx)）。FHWAの事例は架替を含む幅広い工種を対象としており、補修に特化したものではないが、橋梁バンドリングの効果・課題に関する先行研究群として参照できる。国内でも[土木学会ガイドライン（案）2015](https://committees.jsce.or.jp/cmc/node/107)や[群マネ手引き](https://www.mlit.go.jp/sogoseisaku/maintenance/gunmane.html)が補修工事を含む業務の集約発注を論じており、本研究の実務的位置付けを示す根拠となる。

### 1-3. 日本の管理体制の特殊性（日米比較）

日米比較の主要論点：
- 市町村：管理橋梁50〜150橋程度（小規模・技術職員少）
- State DOT：800〜1,300橋程度（専門組織・専任技術者）
- 日本：市町村が約6割の橋梁を管理（道路統計年報）
- 米国：州DOTが主体、市は相対的に少ない

確認要：日米の管理体制比較を示す統計・文献（`日米管理階層比較_勉強`フォルダ内の資料を参照）

## 2. 発注者側の実務負担

### 2-1. 発注者業務の全体像

橋梁補修工事における発注者業務は、調達手続き（入札・契約）だけにとどまらない。[土木学会ガイドライン（案）本編（2015）](https://committees.jsce.or.jp/cmc/node/107) が示すように、点検・評価から計画策定・設計委託・施工監督・記録管理までの一連のサイクルを発注者が担う。

| 業務 | 主な根拠 |
|---|---|
| 長寿命化修繕計画の策定・見直し | 道路法第5条の3・[橋梁定期点検要領](https://www.mlit.go.jp/road/sisaku/yobohozen/yobohozen.html) |
| 定期点検の発注・結果確認 | 道路法施行規則第4条の5の5（5年に1回） |
| 損傷評価・補修優先順位付け | 橋梁定期点検要領（健全性区分I〜IV） |
| 補修対象橋梁の選定・セット決定 | 発注者裁量（発注計画段階） |
| 補修設計の委託・成果確認 | [品確法](https://elaws.e-gov.go.jp/document?lawid=417AC0000000018)（仕様設定責任） |
| 発注計画・入札契約方式の決定 | 品確法・[発注関係事務の運用に関する指針](https://www.mlit.go.jp/tec/tec_reiwaunyoshsishin.html) |
| 仕様書・特記仕様書の作成 | 品確法（適切な仕様設定義務） |
| 積算（設計書・数量計算書） | [土木工事積算基準](https://www.mlit.go.jp/tec/koujisekisan.html) |
| 入札公告・参加資格確認・質疑対応 | [入契法](https://elaws.e-gov.go.jp/document?lawid=412AC0000000101)（情報公開・競争性確保） |
| 入札・開札・落札者決定 | 入契法・地方自治法 |
| 契約締結・変更契約 | [公共工事標準請負契約約款](https://www.mlit.go.jp/totikensangyo/const/content/001857730.pdf) |
| 監督・現場立会・指示 | 品確法（監督責任）・公共工事標準請負契約約款 |
| 設計変更対応・数量確認 | 品確法（適切な設計変更義務） |
| 中間検査・完成検査 | 地方自治法施行令・契約約款 |
| 支払・精算 | 契約約款・会計規則 |
| 台帳更新・施工記録の保管 | 道路法・橋梁定期点検要領 |
| 次サイクルへの反映（計画更新） | 長寿命化修繕計画の定期見直し |

発注者業務の法的基盤は以下の「担い手3法」（2014年・2019年一体改正）が構成する。

**[品確法](https://elaws.e-gov.go.jp/document?lawid=417AC0000000018)（公共工事の品質確保の促進に関する法律）**　
発注者責任を規律する中心的な法律。適切な仕様設定・積算・設計変更・工期設定に加え、ダンピング対策・多様な入札契約方式の活用を規定する。契約集約の可否・方式の選択も発注者責任の範囲に含まれる。

**[建設業法](https://elaws.e-gov.go.jp/document?lawid=324AC0000000100)**　
受注者側を規律するが、発注者の契約設計にも影響する。監理技術者の専任要件（請負額4,000万円以上）は契約規模の上限設計に直結し、バンドリングによる規模拡大がこの要件を発動させるかどうかは実務上の論点である（3章）。

**[入契法](https://elaws.e-gov.go.jp/document?lawid=412AC0000000101)（公共工事の入札及び契約の適正化の促進に関する法律）**　
情報公開・競争性確保・不正排除を規律する。契約規模拡大による競争性低下は入契法の趣旨と緊張関係にあり、地元企業への配慮と広域バンドリングのトレードオフが生じる（4章・7-4章）。

中小規模の地方公共団体では技術職員の不足・業務の多様化・現場離れによる技術力低下により、これらを厳格に運用することが困難な場合があるとされる（[松本ら（2019）](https://www.jice.or.jp/cms/kokudo/pdf/tech/essays/Consideration_on_measures_to_secure_management_capabilities_of_public_works_ordering_organizations.pdf)）。

### 2-2. 入札契約方式の設計構造

公共工事・維持管理業務の入札契約方式は、単一の契約類型として決まるものではなく、複数の設計軸を組み合わせて決定される。[国交省「公共工事の入札契約方式の適用に関するガイドライン」](https://www.mlit.go.jp/tec/content/001475361.pdf)では、事業特性・リスク・発注者体制・施工者の技術力活用の必要性等に応じてこれらを選択するものとして整理されている。つまり、入札契約方式の設計とは「どの契約名を使うか」ではなく、どの業務を、どの範囲で、何年分、誰に、どう競争させ、どう支払うかを組み合わせて決める行為である。

**各設計軸の内容**

**発注範囲（事業プロセスの対象範囲）**

- *施工のみ*：設計はコンサルタントに別途発注し、施工のみを工事会社に発注する。補修工事の現状の標準的形態。
- *ECI方式（Early Contractor Involvement）*：詳細設計段階から施工予定者が参画し、施工性・コストへの意見を反映できる。損傷状況が設計前に確定しにくい橋梁補修への適用余地がある。
- *設計施工一括（DB方式）*：設計と施工を同一者に発注し、責任の一元化・工期短縮が期待できる。競争性確保が課題。
- *維持管理付工事（DBO等）*：施工後の維持管理業務まで含めて一体発注する。長期的な品質確保を狙うが、費用分担・業者評価が複雑になる。

本研究では補修工事の施工（施工のみ形態）に固定する。

**契約単位（工事・業務の発注単位）**

- *単橋*：橋梁1基ごとに発注する最も一般的な形態。小規模・点在する補修需要に対しては発注管理件数が累積する。
- *複数橋（包括発注）*：同一地域内の複数橋梁を1契約でまとめる。[国交省ガイドライン](https://www.mlit.go.jp/tec/content/001475361.pdf)の「包括発注方式」（「同一地域内での複数の種類又は工区の業務・工事を一つの契約により発注する方式」）に対応し、[土木学会ガイドライン（案）](https://committees.jsce.or.jp/cmc/node/107)の「①数量の包括化」と接続する。
- *路線・地域一括*：路線単位または一定エリア単位でまとめる。[宮崎県建設技術推進機構](https://www.qsr.mlit.go.jp/content/900014505.pdf)の地域一括発注や[奈良モデル](https://www.pref.nara.jp/dd.aspx?moduleid=23279&pfromid=60)等の事例がある。
- *複数自治体共同発注*：複数の道路管理者が費用を分担し代表主体が一括発注する。[土木学会ガイドライン（案）](https://committees.jsce.or.jp/cmc/node/107)の「④複数発注者による共同処理」、[群マネ手引き](https://www.mlit.go.jp/report/press/content/001964898.pdf)の広域連携（代行・共同発注・協議会）に対応する。

**〔参考〕土木学会の発注規模拡大4類型**

[土木学会ガイドライン（案）本編（2015）](https://committees.jsce.or.jp/cmc/node/107)は発注規模拡大を以下の4類型で整理する。本研究が扱うバンドリングは主に①・④に相当する。

| 類型 | 内容 | 本研究との関係 |
|---|---|---|
| ① 数量の包括化 | 同種業務の数量をまとめて発注（同工種の補修工事を複数橋でまとめる等） | **主要**（本研究の発注単位集約に直対応） |
| ② 業務の包括化 | 点検・補修設計・工事など異なる業務を一括委託 | 発注範囲軸のECI・DB・長期包括に対応（本研究では固定） |
| ③ 施設の包括化 | 橋梁・道路・河川など異なる施設種別を一括 | 多分野連携の群マネに対応（本研究では対象外） |
| ④ 複数発注者による共同処理 | 複数自治体が共同で発注 | **主要**（広域連携バンドリングに対応） |

**契約期間**

- *単年度*：毎年度契約を締結する標準形態。予算単年度主義と整合するが、毎年度の発注手続きが繰り返される。
- *複数年（債務負担行為）*：複数年度にわたる契約。年度ごとの発注手続きが不要になるが、債務負担行為の議会議決が必要。
- *長期包括*：5年程度の長期で点検・設計・工事を一括委託する形態。多摩市・四国地整等の事例がある。

**支払方式**

- *総価契約*：工事全体の金額を確定して締結する。補修工事では設計変更が頻発しやすく、変更契約の手続きが発生しやすい。
- *単価契約*：単価のみ定め実施数量に応じて精算する。巡回・日常点検等、数量が事前確定しにくい維持管理業務に適する。
- *総価契約単価合意方式*：総価契約でありながら、変更時に備えて主要単価を事前合意しておく。補修工事の設計変更対応として有効。

**入札方式（競争参加者の設定）**

- *一般競争入札*：広く競争参加者を募る原則形態。地域要件・資格要件を設定することで参加資格を絞れる。
- *指名競争入札*：発注者が参加業者を指名する。小規模案件で用いられるが透明性の観点から縮小傾向にある。
- *随意契約*：競争によらず特定業者と契約する。緊急対応・少額案件等で用いられる。
- *JV（共同企業体）*：複数業者が共同受注する形態。地域維持型JVは小規模・多頻度の維持管理工事を地元企業が継続受注するための仕組み。

**落札方式**

- *最低価格落札方式*：価格競争を原則とし最低価格の入札者が落札する。ダンピングリスクがある。
- *総合評価落札方式*：価格と技術提案・施工計画等を総合評価する。維持管理・包括委託で広く用いられる。
- *技術提案・交渉方式*：技術提案を評価し価格交渉を経て契約する。複雑な仕様が事前確定しにくい工事に適する。

**発注単位が発注者の反復事務を決める**

この6軸の中で、本研究と最も直接に関わるのが発注単位である。発注単位が変わると、仕様書作成・積算・公告・契約・監督・検査・変更契約といった発注関係事務の反復回数が変わる。単橋発注であればこれらが橋梁ごとに繰り返されるが、複数橋梁を1契約にまとめると、契約単位で発生する事務が集約される。

ただし、契約単位の集約は発注者負担を一律に削減するものではない。個別橋梁ごとの現地確認・数量確認・品質管理・交通規制・地元調整・出来形確認は残り、工程調整・費用按分・代表主体の管理・複数自治体間協議・受注者の多現場管理は増える可能性がある。

**橋梁補修工事の特性との関係**

橋梁補修工事は対象が既設構造物であるため、施工前に損傷状況や施工条件を完全に把握することが難しい。足場設置後やはつり調査後に損傷範囲が明らかになることも多く、当初仕様・数量と実施数量に差が生じやすい。そのため発注単位の設計は単に契約件数を減らす問題ではなく、変更契約の頻度・監督体制・現場間距離・技術者配置・受注者の施工管理負担を含めた調達設計上の問題となる。

**本研究の位置付け**

本研究が扱うバンドリングは、上記の設計軸のうち「工事・業務の発注単位」に関わる問題である。複数橋梁の補修需要をどの空間範囲で1つの発注ロットとして扱うかを検討するものであり、国交省ガイドラインの「包括発注方式」、土木学会ガイドラインの「数量の包括化」「複数発注者による共同処理」と対応する。入札方式・落札者決定方式・支払方式・契約期間そのものの最適化は扱わず、これらは所与とする。

### 2-3. 契約単位の設計における制度的・実務的留意事項

発注者は契約単位を自由に設定できるわけではなく、法制度や実務上の要請を踏まえて調達設計を行う必要がある。本節では、日本の公的資料で繰り返し示されている主要な設計上の留意事項を整理する。

**2-3-1. 契約規模・包括範囲の設計**

包括化の前提として、まとめる対象の適正な範囲を設定する必要がある。[土木学会ガイドライン（案）](https://committees.jsce.or.jp/cmc/node/107)は包括化が有効に機能する条件として工種・施工時期・地域の類似性を挙げる。[四国地方整備局資料](https://www.skr.mlit.go.jp/etc/R0108_kyoryohosyu.pdf)は橋梁間距離50km程度を実務上の目安として示す。また[包括委託手引き](https://www.mlit.go.jp/sogoseisaku/maintenance/_pdf/houkatsu_tebiki.pdf)は緊急工事・突発対応を包括化に馴染みにくい業務として位置付けており、全件を一括できるわけではないことを示している。距離・工種・緊急性はいずれも「どこまでまとめるか」の設計条件である。

**2-3-2. 競争性の確保**

発注規模をどこまで大きくするかは競争性に直結する。契約規模の拡大は参加可能業者を絞り込み、競争性を低下させるリスクがある。[品確法](https://elaws.e-gov.go.jp/document?lawid=417AC0000000018)は適正な競争の確保を発注者責任の一つとして定め、[入契法](https://elaws.e-gov.go.jp/document?lawid=412AC0000000101)は入札・契約手続きの透明性・公正性を規律する。包括発注方式を採用する場合でも、参加企業数や競争状況を確認しながら発注規模を設定することが求められる。

**2-3-3. 地元企業・地域維持への配慮**

地域をどこまで広げるかは地元企業の参加可能性に影響する。品確法は地域の担い手確保・育成への配慮を発注者の努力義務として規定する。[群マネ手引き](https://www.mlit.go.jp/sogoseisaku/maintenance/_pdf/gunmane_tebiki_honpen2up.pdf)および[包括委託手引き](https://www.mlit.go.jp/sogoseisaku/maintenance/_pdf/houkatsu_tebiki.pdf)では、JV・事業協同組合・域外業者の参画等による受注機会の確保策が論点として挙げられている。

**2-3-4. 広域連携に伴う代表主体・費用分担・議会手続**

何自治体までまとめるかは制度上の連携形態の選択を規定する。複数管理者が共同で発注する場合、以下の制約が伴う（[群マネ手引き](https://www.mlit.go.jp/sogoseisaku/maintenance/_pdf/gunmane_tebiki_honpen2up.pdf)、[国交省 官民連携その3（2024）](https://www.mlit.go.jp/sogoseisaku/kanminrenkei/content/001761547.pdf)）。

| 制約 | 内容 | 本研究との関係 |
|---|---|---|
| 連携形態の選択 | 複数管理者による共同発注では、連携協約・事務委託・一部事務組合・代行等の制度を選択する必要がある。選択する制度によって契約主体・権限分担・責任範囲が異なる。 | 契約単位の実現は制度的な連携形態に依存する。 |
| 費用分担の合意 | 共同発注では各管理者が負担する費用割合や精算方法について事前の合意が必要となる。管理橋梁数・事業量・受益割合等に応じた按分方法を決定しなければならない。 | 広域化しても費用負担の合意が得られなければ実施できない。 |
| 議会手続き | 自治体間で負担金を支出する場合は議会議決等の行政手続きが必要となる。参加自治体が増えるほど意思決定主体が増加し、調整期間が長期化する可能性がある。 | 地域数 \(M\) を増やすほど制度的な実施コストが増加する。 |
| 代表主体の設定 | 協議会や代表自治体が契約主体となる場合、発注事務・契約管理・関係機関調整・履行確認等の業務が代表主体へ集中する。一方で各自治体の個別発注事務は削減される。 | 契約件数が減少しても、負担が消えるのではなく代表主体へ移転する場合がある。 |

このように、複数管理者による共同発注では、契約単位を拡大するほど発注事務の集約が期待される一方で、制度選択・費用分担・意思決定・代表主体への負担集中といった新たな調整コストが生じる。したがって、地域数 \(M\) は数理的な最適化のみで決定できる変数ではなく、制度的実現可能性や自治体間の合意形成を反映した政策パラメータとして解釈することが適切である。

**2-3-5. 本研究との位置付け**

以上のように、契約単位の設計には契約規模・競争性・地域維持・制度的実現可能性等の多様な観点が存在する。本研究では、このうち空間的な契約単位の設計に着目し、発注効率化との関係を定量的に評価する。その他の制度的・市場的条件は、本研究が明示的にモデル化しない実務上の設計条件として、考察に位置付ける（7-4節参照）。

### 2-4. 契約集約による発注者業務の変化

**〔分析の枠組み：比較軸と費用構造〕**

2-1で整理した発注者業務は、契約集約の効果を論じる際に以下の二軸で変化する。

- **比較軸①「バンドリングする/しない」**：橋梁ごとN件に分けて発注する場合と1件にまとめる場合の比較。契約件数の増減が直接現れる。
- **比較軸②「バンドルサイズ」**：一括発注を前提として、何橋をまとめるかの比較（N橋 vs M橋）。契約件数は変わらず、対象橋梁数・施工範囲が変化する。

これらは実際には同時に起きるが、費用への影響の性質が異なる。

- **固定費（比較軸①で削減）**：入札公告・積算・契約締結・完成検査・支払等、1件の契約処理に伴う事務コスト。N件→1件で地域合計として**真に削減**される。
- **変動費（比較軸②で変化）**：監督・現場立会・設計変更対応・台帳更新等、橋梁数・現場条件に比例する業務コスト。対象橋梁数が変わらなければ1件に集約されても地域合計は変わらない（**再配分**）。ただし近接橋梁では動員効率が上がる場合がある。

この枠組みは3章（受注者側）でも同様に適用され、参照する項目が異なる（入札書類・技術者確保・JV組成等）。期待契約件数が固定費削減の代理指標として有効であり、変動費を代理しない理由もここから説明できる（7章）。

| 業務項目        | 契約集約による変化       | 解説                                              | 根拠資料                                                                                                                                                          |
| ----------- | --------------- | ----------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 公告・契約手続     | **○減少**         | 契約件数に比例して発生するため、集約により直接削減される                    | [多摩市（2023）](https://www.mlit.go.jp/sogoseisaku/kanminrenkei/content/001625963.pdf)（p.22）、[宮崎県建設技術推進機構](https://www.qsr.mlit.go.jp/content/900014505.pdf)（p.4） |
| 積算・仕様書作成    | **△一部減少**       | 契約共通部分は集約できる一方、橋梁ごとの数量確認・施工条件の把握は個別に必要となる。      | [四国地方整備局（2019）](https://www.skr.mlit.go.jp/etc/R0108_kyoryohosyu.pdf)（p.18-19）                                                                                |
| 監督・完成検査     | **△性質が変化**      | 契契約管理業務は集約される一方、現場確認・出来形確認等の技術的監督業務は橋梁ごとに必要となる。 | [多摩市（2023）](https://www.mlit.go.jp/sogoseisaku/kanminrenkei/content/001625963.pdf)（p.20）                                                                      |
| 関係機関協議      | **△集約可能だが条件次第** | 契約単位で集約できるものがある一方、広域連携時は協議対象が増加する場合もある          | [群マネ手引き](https://www.mlit.go.jp/report/press/content/001964898.pdf)（p.13）                                                                                     |
| 費用分担調整      | **↑代表主体へ移転**    | 広域連携時のみ発生。各管理者間の按分合意・精算が必要                      | [国交省 官民連携その3（2024）](https://www.mlit.go.jp/sogoseisaku/kanminrenkei/content/001761547.pdf)（p.20-24）                                                           |
| 議会手続・協定     | **↑代表主体へ移転**    | 自治体間の負担金支出には議会議決等が必要。参加者が増えるほど調整期間が長期化          | [群マネ手引き](https://www.mlit.go.jp/report/press/content/001964898.pdf)（p.9）                                                                                      |
| モニタリング・全体管理 | **↑代表主体へ集約**    | 個別自治体の発注事務は削減されるが、代表主体・協議会への管理負担が集中する           | [国交省 官民連携その3（2024）](https://www.mlit.go.jp/sogoseisaku/kanminrenkei/content/001761547.pdf)（p.11-15）                                                           |
| 書類作成（導入初期）  | **↑一時的に増加**     | 新たな包括契約に伴う仕様・基準整備で導入初期は作業量が増える場合がある             | [国交省 官民連携その2（2024）](https://www.mlit.go.jp/sogoseisaku/kanminrenkei/content/001761545.pdf)                                                                    |

なお、ここで削減されるのは契約単位に依存する発注関係事務であり、橋梁ごとの現地確認・施工条件の把握・出来形確認等の技術的業務は依然として必要である。したがって、契約集約は発注者業務全体を削減するものではなく、契約管理業務の集約と一部業務の代表主体への移転をもたらすものと解釈できる。

## 3. 受注者側の実務負担

**本章の目的：** 受注者が契約を履行するために行う業務を記述する。2章の発注者業務と対称的な構造を持ち、3-5で固定費・変動費（比較軸①②）の観点から契約集約との関係を整理する。効果・副作用の評価は4章、期待契約件数の代理指標としての妥当性と限界は7章で扱う。

### 3-1. 入札参加・契約準備

受注者は1件の契約を受注するにあたり、以下の業務を反復する。

| 業務 | 内容 | 根拠 |
|---|---|---|
| 現地調査・事前確認 | 入札前に施工条件・損傷状況・足場設置の可否等を確認する | 公共工事標準請負契約約款 |
| 見積・積算 | 工種・数量・単価を算出し入札金額を積算する | [土木工事積算基準](https://www.mlit.go.jp/tec/koujisekisan.html) |
| 入札書類の作成 | 参加資格確認書類・施工計画書案・技術者届等を作成する | 競争参加資格審査 |
| 技術者確保・配置計画 | 主任技術者・監理技術者・現場代理人の配置を決定する。請負額4,000万円以上では監理技術者の専任が必要 | [建設業法](https://elaws.e-gov.go.jp/document?lawid=324AC0000000100)第26条 |
| 下請調整・資材手配 | 足場業者・専門工種の下請業者・材料調達を事前に調整する | 建設業法（下請規制） |
| 契約手続・施工計画書 | 契約書・請負代金内訳書・工程表・施工計画書を提出する | [公共工事標準請負契約約款](https://www.mlit.go.jp/totikensangyo/const/content/001857730.pdf) |

これらは1契約ごとに反復するため、契約件数が増えるほど受注者の準備・事務コストが累積する。

### 3-2. 施工管理コスト構造（橋梁数・現場条件に依存する業務）

施工段階では、橋梁数・現場条件に応じて発生する業務がある。

[四国地方整備局（2019）](https://www.skr.mlit.go.jp/etc/R0108_kyoryohosyu.pdf) は橋梁補修工事の施工コスト構造について以下を示す。

| コスト項目 | 依存構造 | 内容 |
|---|---|---|
| 直接工事費（施工） | **橋梁数・数量依存** | はつり・断面修復・塗装等の施工量は橋梁の損傷状況に依存し、契約集約では変化しない |
| 共通仮設費（足場・交通規制等） | **現場数・条件依存** | 橋梁ごとに設置が必要。12橋一括算定では1kmグルーピング・1橋個別算定と比較して間接費が過小算定される（p.18-19） |
| 現場管理費（現場代理人・安全管理） | **契約×現場数依存** | 現場代理人は契約ごとに1名必要だが、現場が分散すると移動・管理負担が増加する |
| 資機材回送・現場間移動 | **距離・現場数依存** | 施工箇所が点在するほど機材回送費・交通規制の重複コストが増大する（p.19） |
| 出来形確認・品質管理記録 | **橋梁数依存** | 橋梁ごとに施工記録・出来形確認・試験が必要であり、契約集約では変化しない |
| 施工計画・工程管理 | **契約×現場数依存** | 全体工程表は契約単位で1本だが、現場が分散すると工程調整・連絡調整の複雑さが増す |

また橋梁補修工事では足場設置後やはつり調査後に損傷範囲が確定することが多く、当初数量と実施数量に乖離が生じやすい。設計変更が頻発しやすい点は施工規模にかかわらず橋梁補修の構造的特性である（[土木学会ガイドライン（案）](https://committees.jsce.or.jp/cmc/node/107)）。

### 3-3. JV・共同施工（契約規模拡大への対応策）

契約規模が拡大するにつれて、受注者は単独受注からJV・共同施工への移行を求められる場合がある。これは契約集約に伴う受注者側の適応コストである。

地域維持型JVは維持管理工事に地域の建設企業が継続的に参加できるよう技術者専任要件を緩和した形態であるが（[国交省建設業法関連](https://www.mlit.go.jp/totikensangyo/const/)）、以下の調整コストが発生する。

| JVに伴う業務 | 内容 |
|---|---|
| JV協定・役割分担の合意 | 出資比率・施工担当・技術者配置・利益分配を事前合意する |
| 連帯責任への対応 | 構成員は工事全体について連帯責任を負い、品質管理・瑕疵担保も共同で担う |
| 書類・窓口の一本化 | 発注者への書類提出・連絡調整は代表構成員が担うが、構成員間の情報共有が必要 |
| 技術者・資機材の調整 | 複数企業間で技術者・重機・資材の配置調整が必要となる |

[国交省 官民連携その3（2024）](https://www.mlit.go.jp/sogoseisaku/kanminrenkei/content/001761547.pdf) では、包括委託の試行業務においてJV内の役割分担や資格要件が参画の障壁となった事例が報告されている（p.30-34）。

### 3-4. 不調・不落の発生構造

不調・不落は受注者業務の負担が入札参加インセンティブを下回る状態として発生する。3-1〜3-3の構造から、その発生には二方向の限界がある。

**① 小さすぎる場合（固定費回収不能）**

契約規模が小さい場合、3-1で示した入札参加・契約準備コスト（現地調査・積算・書類・技術者確保等）が受注後の利益で回収できない。[土木学会ガイドライン（案）](https://committees.jsce.or.jp/cmc/node/107) は「発注規模が小さい・労力に見合わない・技術者拘束がある」ことを補修工事の不調要因として挙げる（p.17）。

**② 大きすぎる場合（施工能力・技術者制約）**

契約規模が拡大すると、3-2・3-3で示した技術者配置要件・JV組成・資機材確保・工程調整が参入障壁となる。地域の施工業者が対応できない規模になれば、競争参加者が限定され不落につながる。[四国地方整備局（2019）](https://www.skr.mlit.go.jp/etc/R0108_kyoryohosyu.pdf) が橋梁間距離50km程度を実務目安とした背景にも、遠距離化による施工管理コスト増と参入可能業者の限定がある。

### 3-5. 受注者業務の構造整理

**〔比較軸と費用構造〕**

2章（発注者側）と同じ枠組みを受注者側に適用する。契約集約の効果は以下の二軸で異なる性質を持つ。

- **比較軸①「バンドリングする/しない」**：橋梁ごとN件に分けて発注する場合と1件にまとめる場合の比較。
- **比較軸②「バンドルサイズ」**：一括発注を前提として、何橋をまとめるかの比較（N橋 vs M橋）。

受注者費用の観点では：

- **固定費（比較軸①で削減）**：入札書類作成・技術者確保・JV組成・契約手続等、1件の契約に対して固定的に発生するコスト。N件→1件で地域合計として**真に削減**される。
- **変動費（比較軸②で変化）**：現地調査・積算・施工・出来形確認等、橋梁数・現場条件に応じて発生するコスト。対象橋梁数が変わらなければ、件数を集約しても地域合計は変わらない（**再配分**）。ただし近接橋梁では動員効率が上がり真に削減される場合がある。遠距離では逆に増加する（[四国地方整備局（2019）](https://www.skr.mlit.go.jp/etc/R0108_kyoryohosyu.pdf) の50km目安はこの分岐点を示す）。

なお比較軸①②は実際には同時に発生する（N件を1件にまとめれば、その1件の中の橋梁数がN倍になる）。しかし費用への影響の性質が異なるため、分析上は区別する。この二層構造が、期待契約件数が固定費コストの代理指標として有効であり、変動費を代理しない理由でもある（7章）。

以上を踏まえ、受注者業務を「契約件数依存」「橋梁数・現場条件依存」という軸で分解する。契約集約による増減の評価は4章で行う。

| 業務 | 契約件数依存 | 橋梁数依存 | 主な規定・制約 |
|---|---|---|---|
| 現地調査・事前確認 | ○ | △ | 競争参加資格・仕様書 |
| 見積・積算 | ○ | △ | 土木工事積算基準 |
| 入札書類作成 | ○ | × | 入札制度・参加資格要件 |
| 技術者確保・配置 | ○ | △ | 建設業法第26条（専任要件） |
| 下請調整・資材手配 | ○ | △ | 建設業法（下請規制） |
| 契約手続・施工計画書 | ○ | △ | 公共工事標準請負契約約款 |
| 直接施工（はつり・補修等） | × | ○ | 土木工事標準歩掛 |
| 共通仮設（足場・交通規制） | △ | ○ | 土木工事積算基準 |
| 現場管理（現場代理人・安全） | △ | ○ | 建設業法・労働安全衛生法 |
| 資機材回送・現場間移動 | × | ○ | 積算基準（共通仮設費） |
| 出来形確認・品質管理記録 | × | ○ | 公共工事標準請負契約約款 |
| 完成書類・竣工検査 | ○ | △ | 公共工事標準請負契約約款 |
| JV協定・役割分担調整 | ○ | × | 建設業法（JV取扱要綱） |

○：主として依存する　△：部分的に依存する　×：依存しない

この分解から、受注者業務は「契約件数に依存する業務」（入札参加・契約管理・書類作成等）と「橋梁数・現場条件に依存する業務」（施工・共通仮設・現場管理・出来形確認等）の二層構造を持つことが確認できる。前者は契約集約によって削減される可能性があるが、後者は契約集約の影響を受けにくい。

## 4. 契約バンドリングの実務的効果と副作用

### 4-1. 発注者側への影響

**固定費削減（比較軸①）**

2-1・2-4で整理した発注者業務のうち、契約件数に依存する固定費（入札公告・積算・契約締結・完成検査・支払等）は、契約集約によって地域合計として真に削減される。[多摩市（2023）](https://www.mlit.go.jp/sogoseisaku/kanminrenkei/content/001625963.pdf)では包括委託導入後に発注件数・契約事務の実質的な削減が確認されている（p.22）。[宮崎県建設技術推進機構](https://www.qsr.mlit.go.jp/content/900014505.pdf)の地域一括発注でも、契約管理単位の削減が主要な効果として報告されている（p.4）。

また、小規模自治体にとっては発注件数の削減が技術職員の業務量軽減に直結し、技術力不足の補完につながる（[松本ら（2019）](https://www.jice.or.jp/cms/kokudo/pdf/tech/essays/Consideration_on_measures_to_secure_management_capabilities_of_public_works_ordering_organizations.pdf)）。

**変動費の変化（比較軸②、条件依存）**

監督・現場立会・設計変更対応・台帳更新等の橋梁数依存業務は、対象橋梁数が変わらなければ削減されない。契約を集約しても橋梁ごとの技術的確認業務は残る。ただし近接橋梁をまとめた場合は監督員の動員効率が上がる場合がある。

**代表主体への負担移転（広域連携時）**

複数管理者による共同発注では、各自治体の個別発注事務は削減されるが、費用分担調整・議会手続・モニタリング・連絡調整が代表主体・協議会へ集中する（2-3-4参照）。負担が消えるのではなく移転する点に留意が必要である（[群マネ手引き](https://www.mlit.go.jp/report/press/content/001964898.pdf)・[国交省 官民連携その3（2024）](https://www.mlit.go.jp/sogoseisaku/kanminrenkei/content/001761547.pdf)）。

| 業務 | 変化の方向 | 比較軸 |
|---|---|---|
| 入札公告・積算・契約締結・検査・支払 | **削減**（真の削減） | ① |
| 監督・現場立会・設計変更・台帳更新 | **不変**（再配分）/ 近接なら効率化 | ② |
| 費用分担調整・議会手続・モニタリング | **代表主体へ移転**（広域連携時のみ） | ①に伴う制度コスト |
| 書類・仕様整備（導入初期） | **一時的に増加** | 移行コスト |

### 4-2. 受注者側への影響（3章の受け）

**固定費削減（比較軸①）**

3-1・3-5で整理した受注者業務のうち、入札書類作成・技術者確保・JV組成・契約手続等の固定費は、受注する契約件数が減れば地域全体として真に削減される。小規模・点在する補修工事では、この固定費が利益を圧迫して不調・不落の原因になっていた（[土木学会ガイドライン（案）](https://committees.jsce.or.jp/cmc/node/107) p.17）。契約集約によって1件あたりの採算性が改善し、入札参加インセンティブが高まる可能性がある。

**変動費の変化（比較軸②、近接/遠距離で分岐）**

現地調査・施工・共通仮設・出来形確認等の変動費は、対象橋梁数が変わらなければ地域合計は変化しない。ただし近接橋梁に限れば動員効率・機材回送・交通規制の重複が削減できる。遠距離になるほど逆に増加する（[四国地方整備局（2019）](https://www.skr.mlit.go.jp/etc/R0108_kyoryohosyu.pdf) p.18-19）。この分岐点が50km目安の実務的根拠である。

**JV・規模拡大への適応コスト**

契約規模が拡大すると、3-3で整理したJV組成・役割分担・連帯責任への対応コストが発生する。[国交省 官民連携その3（2024）](https://www.mlit.go.jp/sogoseisaku/kanminrenkei/content/001761547.pdf) ではJV内の役割分担・資格要件が参画障壁となった事例が報告されている（p.30-34）。

| 業務 | 変化の方向 | 比較軸 |
|---|---|---|
| 入札書類・技術者確保・JV組成・契約手続 | **削減**（真の削減） | ① |
| 現地調査・積算・施工・出来形確認 | **不変**（再配分）/ 近接なら効率化 | ② |
| 共通仮設・資機材回送・現場管理 | **近接：削減 / 遠距離：増加** | ② |
| JV協定・役割分担調整 | **増加**（規模拡大に伴う適応コスト） | ①に伴う構造変化 |

### 4-3. 市場・競争への影響

契約集約は入札市場の構造にも影響を与える。2章・3章の業務構造分析では捉えきれない市場レベルの効果である。

**競争性の変化**

契約規模が拡大するほど参加可能な企業が限定され、競争性が低下するリスクがある。[Estache et al.（2011）](https://www.sciencedirect.com/science/article/abs/pii/S095717871000072X) はインフラ調達における規模の経済と競争性低下のトレードオフを指摘する（p.2-4）。入契法が定める競争性確保の観点（2-3-2参照）とも緊張関係にある。

**地元企業・担い手への影響**

契約規模・JV要件・広域化によって地域の中小企業が参加しにくくなる可能性がある。[国交省 官民連携その3（2024）](https://www.mlit.go.jp/sogoseisaku/kanminrenkei/content/001761547.pdf) および [FHWA Lessons Learned](https://www.fhwa.dot.gov/ipd/alternative_project_delivery/defined/bundled_facilities/lessons_learned.aspx) では、地元企業の参入機会確保が包括発注の課題として繰り返し指摘されている。

**不調・不落の変化**

3-4で整理した通り、不調・不落には「小さすぎる（固定費回収不能）」と「大きすぎる（施工能力制約）」の二方向の限界がある。契約集約は前者を緩和する一方で、後者のリスクを高める可能性がある。適正なバンドルサイズの設計が競争性と採算性の両立に不可欠である。

## 5. 契約論・調達論からみたバンドリング

**本章の目的：** 2〜4章で整理した実務的構造（発注者・受注者業務の固定費/変動費の二層性、競争性トレードオフ）を、契約論・調達論の言語で位置付ける。これにより、7章で採用する「期待契約件数の最小化」という目的関数が、公共調達論として妥当な代理指標であることを示す。本章は新たな実務を説明する章ではなく、**実務 → 理論 → 数理モデル** をつなぐ章として機能する。

### 5-1. 契約単位設計としてのバンドリング

公共調達における **Contract Bundling**（契約バンドリング）とは、複数の財・サービス・工事を一つの契約単位にまとめて発注することを指し、反対に **Unbundling** は契約を小さなロットに分割することである。契約単位の設計（Contract Lot Design）は、発注者の管理負担、受注者の参加可能性、競争性、施工効率を同時に規定する制度設計問題である（Dimitri, Piga and Spagnolo eds., *Handbook of Procurement*）。

契約ロット設計は、入札方式（一般競争・指名競争等）や報酬方式（総価・単価・性能規定等）を選ぶ前段階で、**そもそも何を一つの調達単位として市場に出すかを決める設計問題**である（Tadelis 2012; 2-2参照）。入札方式の選択はロット設計が先に決まって初めて意味を持つ。

橋梁補修工事のバンドリングは、複数橋梁の同時施工計画ではなく、この意味での **公共調達における契約ロット設計問題** として位置付けられる。1橋梁ごとに発注するか、路線単位・地域単位・自治体横断単位で束ねるかは、施工計画の問題ではなく、誰が、何件の契約で、どの範囲を発注するかという調達設計の問題である。本研究は、地域単位の契約ロット設計に焦点を当て、対象橋梁数・地域設定が期待契約件数に与える影響を評価する。

代表的先行研究として、[Estache and Iimi（2011）](https://www.sciencedirect.com/science/article/abs/pii/S095717871000072X)は水道・下水プロジェクトの調達データを用い、契約規模の拡大による規模・範囲の経済と競争性低下のトレードオフをインフラ調達の実証問題として提示した。本研究は同じ Contract Lot Design の文脈に位置するが、焦点を価格形成ではなく発注者側の固定的取引費用に置く点に独自性がある。

### 5-2. 取引費用と期待契約件数

Transaction Cost Economics（取引費用経済学）は、Coaseの「企業の本質」（1937）に端を発し、Williamsonによって体系化された理論である（Williamson 1979, 1985）。市場取引には価格のほかに、探索・交渉・契約締結・監視・履行確認・紛争処理の費用が伴い、これらの取引費用をどう抑制するかが取引単位の設計問題であるとされる。

公共工事調達では、各契約に以下の行政プロセスが付随する。これらは工事数量に完全比例する変動費ではなく、**契約件数に応じて反復的に発生する固定的取引費用**である。

| 取引費用の種類 | 公共調達での内容 | 橋梁補修工事での対応 |
|---|---|---|
| 情報収集費用 | 損傷状況・数量・施工条件の把握 | 橋梁点検結果・損傷程度・交通条件の整理 |
| 入札費用 | 公告・積算・仕様書作成・質疑応答 | 補修工事ごとの発注事務 |
| 契約締結費用 | 契約書・保証・保険 | 補修工事契約の締結 |
| 監督費用 | 工程・品質・出来形確認 | 現場監督・補修数量確認 |
| 契約管理費用 | 設計変更・支払管理 | 数量変更・変更契約対応 |
| 再交渉費用 | 追加費用協議・紛争処理 | 施工中判明損傷への対応 |

この理論的整理は、日本の実務における発注者責任論とも対応する。国土交通省の発注者責任懇談会は、発注者責任を「公正さを確保しつつ良質なモノを低廉な価格でタイムリーに調達し提供する責任」と定義し、発注者業務として「工事内容評価、入札契約方式と企業選定、技術審査、監督、検査、支払、工事実績評価」を整理している。これらはWilliamson的な意味での取引費用の各段階に対応しており、抽象的な理論概念ではなく、日本の発注者責任論において発注者が担うべき業務として制度的に確認されている。

この観点から、**本研究の期待契約件数は、契約ごとに反復する固定的取引費用の代理指標**として解釈できる。契約件数を減らすことは、発注者が繰り返し負担する調達プロセスの固定費用を削減することに対応する。ただし、Williamsonの取引費用経済学では生産費用との合計を最小化することが重要であるため、本研究は取引費用側のうち契約件数に比例する固定的調達費用に焦点を絞っていることを明記する（施工費・変動費は扱わない）。

### 5-3. 規模の経済と競争性低下のトレードオフ

契約をまとめることの便益は、**規模の経済**（Economies of Scale）として説明できる。同種工事の集約により、発注者側の固定的取引費用が件数分散し、受注者側でも資機材動員・人員配置・現場管理の固定費が分散される。

ただし本研究が扱う空間的集約の効果は、純粋な規模の経済よりも**密度の経済**（Economies of Density）に近い側面を持つ。近接橋梁をまとめることで動員効率・資機材回送・交通規制の重複が削減されるが、この効果は橋梁間距離に強く依存し、遠距離になると逆転する（4章、[四国地方整備局（2019）](https://www.skr.mlit.go.jp/etc/R0108_kyoryohosyu.pdf)）。したがって、本研究の空間クラスタリングは密度の経済を補助的に反映するが、直接の最適化対象は施工側の効率ではなく契約件数である。

一方、契約規模拡大の費用として**競争性低下**がある。契約規模が拡大すると参加可能企業の資金力・技術力・実績要件が高まり、入札参加者数が減少して競争圧力が弱まる可能性がある。[Estache and Iimi（2011）](https://www.sciencedirect.com/science/article/abs/pii/S095717871000072X)はこの規模の経済と競争性のトレードオフをインフラ調達において実証的に示した（p.2-4）。日本では[入契法](https://elaws.e-gov.go.jp/document?lawid=412AC0000000101)が競争性確保を規律しており、地元企業参加・担い手確保の観点（[品確法](https://elaws.e-gov.go.jp/document?lawid=417AC0000000018)）とも緊張関係にある（2-3-2・4-3参照）。

本研究はこのトレードオフのうち、**発注者側の固定的取引費用削減（便益側）に焦点を当て、競争性への影響（費用側）は直接扱わない**。これは研究の限界として7章・8章で明示する。

### 5-4. 橋梁補修工事の不完備契約性

Incomplete Contract Theory（不完備契約理論）は、将来起こりうるすべての状態を契約に明記することは不可能であるという前提から出発する（Grossman and Hart 1986; Hart and Moore 1990）。不確実性が高い取引では、契約後の設計変更・再交渉・残余的意思決定権が重要になる。

橋梁補修工事はこの意味で**不完備契約性が高い典型的な公共工事**である。

| 橋梁補修の特徴 | 不完備契約理論での解釈 |
|---|---|
| 損傷が事前に完全には分からない | 将来状態を契約に完全記述できない |
| 数量変更が多い | 契約時点の数量は暫定的で、履行過程で再交渉が生じる |
| 設計変更が頻発する | 新たな情報が判明し、契約内容の適応が必要になる |
| 現場条件に依存する | 状態依存的な契約条項を完全に設計しにくい |

土木維持管理分野においても同様の整理がある。石ら（2019）は、性能規定型維持管理契約において、性能非規定要因を契約条項として完全に記述できないため不完備契約にならざるを得ないと指摘する。また、短期契約の繰り返しでは異時点間のペイオフ外部性・逆選抜・モラルハザードが発生し得るとして、維持管理契約における不完備性を実務的に裏付けている。橋梁補修に直接特化した研究ではないが、**維持管理契約は契約時点で全ての状態・品質要因を記述しきれない**という命題の根拠として援用できる。

この特徴から、橋梁補修工事のバンドリングは単なる契約件数削減ではなく、**複数の不確実な補修需要を一つの契約ガバナンスの中で扱う設計問題**でもある。複数橋梁をまとめることで契約管理の窓口は減るが、契約内部での設計変更・数量変更・再交渉の複雑性は増える可能性がある。本研究では契約内部で発生するこれらの再交渉費用は明示的にモデル化しない。

### 5-5. 共同発注と複数主体間調整

複数自治体が共同で橋梁補修を発注する場合、契約理論上は**Multi-principal problem**（複数主体問題）として整理できる（2-3-4参照）。通常の調達モデルでは単一発注者が受注者に委託するが、共同発注では発注者側に複数の主体が存在し、費用配分・意思決定権限・監督責任の調整問題が発生する。

本研究の地域クラスタリングは共同発注の潜在的な空間単位を示すが、費用分担・制度形態・意思決定の設計は扱わない。地域数Mは数理的最適化変数ではなく、制度的実現可能性を反映した政策パラメータとして扱う（2-3-4・7章）。

### 5-6. 本研究への含意

以上の理論的整理から、本研究の位置付けは次のように確定する。

> 本研究は、橋梁補修工事を対象とした Contract Lot Design の研究として位置付けられる。2〜4章で整理した発注者・受注者業務の固定費/変動費の二層構造は、取引費用経済学における「契約件数に比例する固定的調達費用」として理論化できる。期待契約件数はこの固定的取引費用の代理指標であり、その最小化が本研究の目的関数となる。マルコフ劣化モデルにより確率的に発生する補修需要を生成し、空間クラスタリングにより地域単位の契約ロットを形成することで、契約単位設計が調達効率に与える影響を定量的に評価する。

本研究が直接扱わない要素は以下の通りであり、これらは今後の課題として位置付ける。

| 扱わない要素 | 理論上の位置付け |
|---|---|
| 入札参加者数・落札価格 | Auction Theory / 競争性 |
| 施工費・共通仮設費 | 生産費用（橋梁数依存の変動費） |
| 設計変更・再交渉費用 | Incomplete Contract（不完備性費用） |
| 自治体間の費用配分・制度設計 | Multi-principal problem |

## 6. 橋梁・道路バンドリングの既往研究

### 6-1. 劣化モデルに基づく橋梁ネットワーク維持管理研究

橋梁維持管理の最適化研究は、マルコフ連鎖・確率的動的計画法（StDP）・ライフサイクルコスト最適化を柱として発展してきた。これらの研究群は橋梁の劣化過程を確率的に内生化し、補修行為・補修時期の最適化を行うが、公共調達上の契約ロット設計は最適化対象としない。

**Hadjidemetriou, Xie & Parlikad（2020）** は本研究と最も近い先行研究である。連続時間マルコフ連鎖（CTMC）により橋梁要素の劣化を確率的にモデル化し、shared setup costを持つgroup maintenanceを定式化した（[Transportation Research Record](https://doi.org/10.1177/0361198120912226)）。同研究のグループ化は「同一システム内の複数コンポーネントを同時補修することで段取り費を按分する」という構図であり、橋梁間の補修需要を束ねてどう公共調達の契約とするかという意思決定——入札ロット、発注主体、契約件数——は扱わない。すなわち、確率的劣化は内生化されているが、契約ロット設計は外生または対象外である。

**Frangopol & Liu（2007）** は確率的動的計画法を橋梁ネットワークに適用し、各橋梁の補修行為選択と総期待費用最小化を定式化した（[ASCE Journal of Structural Engineering](https://doi.org/10.1061/(ASCE)0733-9445(2007)133:12(1772))）。**Orcesi & Cremona（2010）** は信頼性理論ベースでPareto最適化を提示し（[Reliability Engineering & System Safety](https://doi.org/10.1016/j.ress.2010.06.013)）、**Wu et al.（2017）** はsemi-Markov過程を用いたライフサイクルコスト最適化を行う（[Applied Mathematical Modelling](https://doi.org/10.1016/j.apm.2016.11.029)）。いずれも補修需要を確率的に内生化するが、cross-bridge contract aggregationは扱わない。

**Bensoussan et al.（2026）** はsemi-Markov MDPにより大規模構造物フリートの確率的維持管理を定式化し、fleet全体でshared setup costを持つgroup maintenanceをMDPとして扱う（Dynamic Games and Applications、最新研究のため今後の査読状況を要確認）。Hadjidemetriou（2020）と類似の枠組みだが、契約ロット設計という意味での入札・調達の意思決定変数は含まれない。

| 文献 | 確率劣化モデル | Shared Setup Cost | 補修行為選択 | 契約ロット設計 |
|---|---|---|---|---|
| Hadjidemetriou 2020 | CTMC | あり（同一システム内） | あり | なし |
| Frangopol & Liu 2007 | Markov / StDP | なし | あり | なし |
| Orcesi & Cremona 2010 | Markov / Reliability | なし | あり | なし |
| Wu et al. 2017 | semi-Markov | なし | あり | なし |
| Bensoussan et al. 2026 | semi-Markov MDP | あり（fleet全体） | あり | なし |

### 6-2. インフラ・道路・橋梁の契約バンドリング研究

インフラ調達における契約バンドリング研究は、所与のプロジェクト集合をいかに束ねるかを扱う。これらの研究群は契約ロット設計を内生化するが、橋梁の確率的劣化から補修需要が発生する過程は外生的な入力として扱う。

**Miralinaghi et al.（2022）** はroad agencyと道路利用者の二層最適化問題として契約バンドリングを定式化し（[Computer-Aided Civil and Infrastructure Engineering](https://doi.org/10.1111/mice.12740)）、同一リンク内の複数プロジェクトをどう束ねるかをバイナリ変数で扱う。リンク路面状態の確率推移はexogenousなschedule inputとして扱われ、stochastic deteriorationは内生化されない。また、バンドルは同一リンク内に限定されており、橋梁間・路線間をまたぐcross-link bundlingは対象外である。

**Qiao et al.（2018）** はINDOT橋梁プロジェクト（1,997件、715契約、9年間）の経験データを用いてbundlingパターンを分析し、mob cost削減と調達overhead削減を重回帰で実証した（[Transportation Research Record](https://doi.org/10.1177/0361198118797804)）。理論的な最適lot sizeの導出ではなく、確率的劣化モデルは用いない。**Qiao et al.（2021）** はbundling政策が入札者数と入札価格に与える影響を分析し、規模の経済と競争性低下のdual effectを提示した（[Transportation Research Part A](https://doi.org/10.1016/j.tra.2021.01.006)）。**Qiao et al.（2026）** は道路プロジェクトバンドリングをcombinatorial optimizationとして定式化し、greedy heuristicで最適bundle戦略を探索するが、対象プロジェクト集合はexogenous inputである（Computer-Aided Civil and Infrastructure Engineering、最新研究のため査読状況を要確認）。

**[FHWA Lessons Learned from State Bridge Bundling Programs](https://www.fhwa.dot.gov/ipd/alternative_project_delivery/defined/bundled_facilities/lessons_learned.aspx)** は設計・施工一括（Design-Build）方式によるbridge bundleがdeliveryを短縮しmob costを削減した事例を報告するが、補修需要をstochastic deteriorationから内生的に導出する数理フレームワークではない。

| 文献 | Bundle単位 | 調達・契約の内生化 | Stochastic劣化内生化 | Lot Size最適化 |
|---|---|---|---|---|
| Miralinaghi 2022 | 同一リンク内プロジェクト | 二層最適化 | なし | 部分的（リンク内） |
| Qiao 2018 | 橋梁＋舗装プロジェクト | 経験的回帰分析 | なし | 経験分析 |
| Qiao 2021 | highway bundle | 入札行動分析 | なし | 経験分析 |
| Qiao 2026 | 道路プロジェクト | combinatorial heuristic | なし | heuristic |
| FHWA | bridge bundle | 記述的事例 | なし | ルールベース |

### 6-3. District Optimization / Territory Design 研究

District Optimization（地区最適化）は、地理的基本単位（basic units）をより大きな地理的クラスタ（districts）に集約する問題の総称である。[Kalcsics（2019）](https://ideas.repec.org/h/spr/sprchp/978-3-030-32177-2_25.html)は Springer の Location Science ハンドブックにおいて、これを「compactness・contiguity・balance の3制約のもとで basic units を districts に集約する問題」と定義し、政治区割り（Political Districting）・配送地区設計（Territory Design）・公共サービス地区割（Service Districting）の3系統が派生的サブ問題として発展してきたことを整理している。

**3つの代表制約**：既往研究でほぼ共通して採用される制約は、contiguity（連続性：地区内が地理的に連結していること）、compactness（コンパクト性：地区の形状が歪でないこと）、workload balance（負荷均衡：各地区の需要量・作業量が均等であること）の3つである。[Bozkaya et al.（2003）](https://www.sciencedirect.com/science/article/abs/pii/S0377221701003800)はタブー探索でこれらをhardに実装し、[Sandoval et al.（2020）](https://www.sciencedirect.com/science/article/abs/pii/S095741741930867X)は厳密MIPにcut-generation戦略でcontiguityを組み込む。

**周辺問題との差異**：Clustering は contiguity を強制せず形状制約を持たない。VRP は既に確定した地域の内部で車両経路を引く問題であり、bundle設計後の問題に相当する。Facility Location は新規拠点の配置問題であり、既存オブジェクトの集約とは構造が異なる。District Optimization は「既存の地理単位をどう集約するか」という点で、これら3問題と明確に区別される。

**本研究との対応**：Kalcsicsの枠組みに照らすと、本研究では橋梁がbasic unit、契約ロットがdistrictに対応する。contiguityは「工事現場の地理的近接性」、compactnessは「動員・機材回送の効率」、workload balanceは「補修需要量・コストの均衡」として読み替えることができ、定式化の骨格を接合できる。橋梁補修に最も近い応用類型はService Districting系列であり、[Camacho-Collados et al.（2015）](https://www.sciencedirect.com/science/article/abs/pii/S0377221715004130)の警察管区最適化や[Assis et al.（2014）](https://www.sciencedirect.com/science/article/pii/S0305054813002086)の電力メータ巡回地区設計と問題構造を共有する。

**ただし**、既存のDistricting研究はいずれも公共調達上の契約件数・入札単位・確率的劣化需要を同時には最適化対象としない。橋梁のbundle化を確率的補修需要と接続して組合せ最適化として定式化した学術研究は、確認した範囲では極めて限定的である。

### 6-4. 研究ギャップ：確率的補修需要・地域分割・契約ロット設計の断絶

以上の整理から、既往研究は三系統に分類できる。

| 研究群 | 需要発生 | 補修行為・時期 | 地域分割 | 契約ロット設計 |
|---|---|---|---|---|
| 劣化モデル系 | 内生 | 内生 | 対象外 | 対象外 |
| 契約バンドリング系 | 外生・所与 | 外生または所与 | 部分的 | 内生 |
| District Optimization系 | 外生・所与 | 対象外 | 内生 | 対象外 |
| 本研究 | 内生 | 内生（補修需要として） | 内生 | 内生 |

既往研究は、劣化モデル系、契約バンドリング系、district optimization系に分かれて発展してきた。劣化モデル系は補修需要を内生化するが契約ロットを扱わず、契約バンドリング系はロット設計を扱うが需要は所与であり、district optimization系は空間分割を扱うが確率的劣化需要や公共調達上の契約件数とは接続していない。本研究はこれら三者を接続し、確率的に発生する橋梁補修需要を地域単位の契約ロットとして設計する点に新規性がある。

## 7. 本研究の数理モデル

### 7-1. 最適化問題の定義

本研究では、N橋の橋梁群を**地域分割問題（Districting Problem）**として定式化し、橋梁をM個の地域へ分割して期待契約件数を最小化する。なお本研究では、「地域」と「契約ロット」を同義として用いる。

問題は2段階の構造を持つ。第1段階では橋梁群をM地域へ分割する（確定的な設計変数）。第2段階では、マルコフ劣化過程から各橋梁の補修需要が確率的に発生し、需要が1件でも生じた地域で1件の契約が締結される（確率的）。目的は、この2段階構造のもとで期待契約件数 $E[C]$ を最小化する地域分割を求めることである。

**分析フレームワーク：**

```
橋梁データ（位置・劣化状態）
          ↓
    マルコフ劣化モデル
          ↓
    補修需要の確率的生成
          ↓
      地域分割（設計）
          ↓
    契約ロット形成（M地域）
          ↓
    期待契約件数 E[C] の評価
```

数式による正式な定式化は 7-6 で行う。

### 7-2. 目的関数：期待契約件数

5章で整理したように、本研究では契約件数を固定的取引費用の代理指標と位置付ける。取引費用経済学（Williamson 1985）の枠組みでは、公告・積算・契約・監督・検査・支払という調達プロセスの各ステップが契約件数に比例して発生する固定費用に相当し、Contract Lot Design研究（Tadelis 2012; Estache & Iimi 2011）はこの費用を最小化する発注ロット設計を調達上の政策変数として扱う。実務においても、[多摩市（2023）](https://www.mlit.go.jp/sogoseisaku/kanminrenkei/content/001625963.pdf)・[宮崎県建設技術推進機構](https://www.qsr.mlit.go.jp/content/900014505.pdf)・[土木学会ガイドライン（案）（2015）](https://committees.jsce.or.jp/cmc/node/107)のいずれも、契約・発注ロットの集約が発注者の手続き負担軽減に直結することを確認している。

**目的関数：**

$$\min_{x} \; E[C] = \min_{x} \; \sum_{j=1}^{M} E\!\left[\mathbf{1}\{\text{地域}j\text{に補修需要あり}\}
\right]$$

$E[C]$ は各期における地域ごとの契約件数の期待値の総和であり、マルコフ劣化過程から決まる各橋梁の補修需要発生確率に依存する。すなわち本研究では、契約件数そのものを最適化するのではなく、契約件数を固定的調達費用の代理指標として扱い、その期待値 $E[C]$ を最小化する。

### 7-3. 空間制約の定式化

6章で整理したDistrict Optimization（Kalcsics 2019）では、地域分割問題の空間制約として **contiguity**（連続性）と **compactness**（コンパクト性）の2つが代表的に採用される。橋梁補修への適用を考えると、contiguity（地域内が地理的に連結していること）より**施工可能な物理的距離**の方が本質的な制約となる。飛び地であっても一定距離圏内なら同一ロットとして施工可能であり、連結していても距離が大きければ動員効率を損なう。したがって本研究では、**contiguityは明示的制約として課さず**、距離制約（$D \leq \varepsilon$）により実務上の近接性を担保する。compactnessは各地域内の**最大橋梁間距離** $D$ として定式化する。

**重み付き目的関数からε制約法へ（設計思想）：**

空間制約をどう扱うかを決めるにあたり、次の思考過程を経た。

出発点として、距離 $D$ は目的関数に組み込める変数である。自然な多目的定式化は次の形をとる：

$$\min_{x} \; \alpha \cdot E[C] + \beta \cdot D$$

距離は理論上は契約件数とのトレードオフを構成する目的関数の一要素と考えられ、6章で整理したDistrict Optimizationにおいてもcompactnessとして目的関数に組み込まれることが多い。しかし本研究では、(1) 重み係数 $\alpha$・$\beta$ の客観的な設定が困難であること、(2) 実務上も地域内距離には一定の管理目安（[四国地方整備局（2019）](https://www.skr.mlit.go.jp/etc/R0108_kyoryohosyu.pdf)が示す「50km程度」）が存在することから、距離をε制約として扱う定式化を採用する：

$$D \leq \varepsilon$$

```
自然な出発点：2目的問題
          ↓
    min αE[C] + βD
          ↓
  ① 重み係数の客観的設定が困難
  ② 実務目安：50km程度（四国地方整備局）
          ↓
  本研究の選択：ε制約法を採用
          D ≤ ε
          ↓
  目的関数を min E[C] に単純化
```

εをシナリオパラメータとして変化させることで、「どの程度の広域化を許容すると期待契約件数がどう変わるか」という感度分析としても解釈できる。本研究では $\varepsilon = 50\text{km}$（直線距離）を基準シナリオとし、直線距離は道路距離の近似指標として用いる（精緻化の余地として残す）。

**補足：重み付き目的関数との関係**

ε制約法と重み付き目的関数法（$\min \alpha E[C] + \beta D$）の関係は、問題の凸性に依存する。連続凸問題では両者は等価なPareto解集合を生成するが、本研究のような二値整数計画ではパレートフロンティアが離散的・非凸になりうるため、重み付き法は非凸な部分の解を拾えない場合がある。ε制約法はεを網羅的に変化させることでそうした解も原理的に発見できる点で、整数計画への適用上の優位がある。

すなわち両定式化の違いは計算結果の違いではなく、パラメータの解釈の違いである。重み付き目的関数では単位が異なる $E[C]$（件数）と $D$（km）の相対重みを設定する必要があり、その客観的な根拠を示すのが難しい。一方、ε制約法では距離上限を「km」として直接設定できるため、実務目安（四国地方整備局の50km）をそのままパラメータに落とし込めるという利点がある。本研究がε制約形式を採用する主な理由はこの解釈容易性にある。

### 7-4. 地域数 $M$ の設定

$M$ は最適化変数として数理的に解くのではなく、**制度的・政策的に実現可能な連携形態に対応するシナリオ**として設定する。

> $M = 1$（全橋梁を一括）、$M = 2 \sim 5$（広域連携単位）、$M = N$（個別発注）といった複数シナリオを設定し、各シナリオ下での最適分割を求めて期待契約件数を比較する。

現実の調達制度では連携体制・組織設計・財政上の理由により地域数が事前に制約される場合が多く、連続的な最適化よりもシナリオ比較の方が政策含意を引き出しやすい。

### 7-5. モデル化の範囲と限界

本研究は**契約ロット設計問題**の定式化を目的とするため、以下の要素はモデルの対象外とする。これらは契約ロット設計の「上流」（調達政策）ではなく「下流」（施工市場・建設生産）に位置する問題である。

| 対象外の要素 | 除外理由 |
|---|---|
| 施工費・直接工事費 | 規模の経済や共通仮設費は契約ロット設計だけで決まらない |
| 入札競争性・参加者数 | 契約規模拡大の競争性への影響は別途分析が必要 |
| JV構成・技術者配置 | 施工市場・建設生産の問題であり契約ロット設計の後段 |
| 供給制約・担い手不足 | 地域別施工業者数等のデータ取得が困難 |

### 7-6. 数理モデルの全体像

以上を統合して、本研究の最適化問題を形式的に定義する。

**意思決定変数：**

$$x_{ij} \in \{0, 1\} \quad (i = 1, \ldots, N; \; j = 1, \ldots, M)$$

$x_{ij} = 1$：橋梁 $i$ を地域 $j$ に割り当てる。

**最適化問題：**

なお、以下では**橋梁間の補修需要発生は独立**と仮定する（この仮定により地域内の需要発生確率が積の形で計算できる）。

$$\min_{x} \; E[C] = \sum_{j=1}^{M} \left(1 - \prod_{i:\, x_{ij}=1}(1 - p_i)\right)$$

ここで $p_i$ は橋梁 $i$ の補修需要発生確率（マルコフ劣化過程から算出）、橋梁間の需要発生は独立と仮定する。

**制約条件：**

$$\sum_{j=1}^{M} x_{ij} = 1 \quad \forall i \qquad \text{（各橋梁はちょうど1地域に割当）}$$

$$\sum_{i=1}^{N} x_{ij} \geq 1 \quad \forall j \qquad \text{（各地域に少なくとも1橋梁）}$$

$$\max_{\substack{i,i':\ x_{ij}=x_{i'j}=1}} d(i, i') \leq \varepsilon \quad \forall j \qquad \text{（地域内最大距離制約）}$$

$$x_{ij} \in \{0, 1\} \quad \forall i, j$$

**本研究の位置付け（まとめ）：**

本研究は、マルコフ劣化モデルで補修需要を内生的に生成し（劣化モデル系の貢献）、その需要を地域単位のDistrict Optimization問題として契約ロット設計に接続する（契約バンドリング系・District Optimization系の貢献）点で、6章で整理した三研究群の交差領域に位置する。

## 8. 本研究の位置付けと新規性

### 8-1. 本研究の位置付け

6章で整理したように、既往研究は三系統に分かれて発展してきた。劣化モデル研究群は補修需要を確率的に内生化するが、公共調達上の契約ロット設計を扱わない。契約バンドリング研究群はロット設計を最適化するが、需要は所与の外生入力として扱う。District Optimization研究群は空間分割を組合せ最適化として定式化するが、確率的な劣化需要や公共調達上の契約件数とは接続していない。

```
劣化モデル研究群              契約バンドリング研究群        District Optimization研究群
（Hadjidemetriou 2020等）     （Qiao 2018/2021等）          （Kalcsics 2019等）
        ↓                            ↓                              ↓
  補修需要の内生化：○          契約ロット設計：○             空間分割最適化：○
  契約ロット設計：✗            補修需要の内生化：✗           確率的劣化需要：✗
  空間分割最適化：✗            空間分割最適化：△             公共調達との接続：✗

                  ↘                  ↓                  ↙
                        【本研究】
              補修需要の内生化 × 地域分割（Districting）× 契約ロット設計
                        を統合した定式化
```

本研究はこれら三者の交差領域に位置し、マルコフ劣化過程により補修需要を確率的に生成し、その需要を地域単位のDistricting Problemとして契約ロット設計に接続する。これは、補修需要と契約ロットを明示的に統合した定式化として、確認した範囲では先行研究にない試みである。

### 8-2. 本研究の新規性

本研究の新規性は以下の3点に整理できる。いずれも「先行研究では〇〇できなかった／していなかった」という研究ギャップとの対比として記述する。

**（1）問題設定の新規性：三研究領域の統合定式化**

劣化モデル研究群は補修需要を確率的に内生化するが契約ロット設計を扱わない。契約バンドリング研究群はロット設計を最適化するが補修需要は所与の外生入力として扱う。District Optimization研究群は空間分割を定式化するが確率的劣化需要や公共調達上の契約件数とは接続していない（6-4参照）。この三系統は断絶したまま発展してきた。本研究はこれらを一つの最適化問題として定式化した点に新規性がある。確率的に発生する橋梁補修需要を地域単位の契約ロットとして設計するという問題の立て方は、確認した範囲では先行研究にない。

**（2）方法論の新規性：橋梁劣化データから調達評価への接続**

既往の劣化モデル研究（Hadjidemetriou 2020等）は補修行為選択とライフサイクルコストを最適化するが、期待契約件数という調達側の指標には接続しない。本研究は「点検データ → マルコフ遷移確率の推定 → 橋梁ごとの補修需要発生確率 $p_i$ → 地域単位の期待契約件数 $E[C]$」という変換パイプラインを構築した。すなわち工学的観測量（橋梁の劣化状態）を出発点として調達政策の評価指標（期待契約件数）を直接導出する枠組みであり、この接続自体が先行研究にない。具体的には、地域単位の期待契約件数を

$$E[C] = \sum_{j=1}^{M} \left(1 - \prod_{i:\, x_{ij}=1}(1 - p_i)\right)$$

として定式化した。取引費用経済学（Williamson 1985）・Contract Lot Design（Tadelis 2012; Estache & Iimi 2011）を理論的根拠として、5章で整理した「契約件数が固定的取引費用の代理指標である」という命題と接続している点も本手法の特徴である。

**（3）実証・政策的貢献：シナリオ分析による定量評価**

先行研究における橋梁バンドリングの政策評価は経験的回帰分析（Qiao 2018）または記述的事例研究（FHWA）に留まり、地域分割・ロットサイズ・距離制約と期待契約件数の関係を確率的モデルから定量化した研究は確認されない。本研究は実橋梁データに基づくマルコフ遷移確率を入力として、地域数 $M$・距離制約 $\varepsilon$・ロットサイズの変化が期待契約件数に与える影響を定量化した。「この自治体において $M=3$、$\varepsilon=50\text{km}$ ではどの程度の契約件数削減が期待できるか」という問いに定量的に回答できる枠組みを示した点が、既往研究にはない政策的含意を持つ。また、本研究により、橋梁数・一括契約橋梁数・地域数・距離制約と期待契約件数との関係を実データに基づいて体系的に評価できることを示した。これにより、単一シナリオの点推定にとどまらず、期待契約件数の振る舞いそのものを明らかにした点が本研究の実証的貢献の核心である。

### 8-3. 実務的貢献

学術的貢献に加えて、本研究は実務上の意思決定支援に直結する枠組みを提供する。

現状の橋梁補修発注は、行政地区境界や担当者の経験則に基づくヒューリスティックスによって契約ロットが決定されることが多い。本研究の枠組みでは、

- 地域数 $M$（広域連携の規模）
- 距離制約 $\varepsilon$（管理可能距離の上限）
- 橋梁の劣化状態分布（マルコフ遷移確率として入力）

を変更しながら期待契約件数を事前評価でき、自治体ごとに最適なロット設計を比較検討できる。これは従来の「経験則によるロット設計」から「確率的補修需要に基づく定量的ロット設計」への転換を可能にする政策シミュレーションの枠組みである。

### 8-4. 今後の展望

本研究は契約ロット設計問題の定式化に焦点を絞るため、以下の要素はモデルの対象外とした。これらは今後の研究拡張として位置付けられる。

| 拡張項目 | 内容 |
|---|---|
| 施工費・共通仮設費 | 規模の経済や動員コストを目的関数に組み込む |
| 入札競争性 | 契約規模拡大が参入業者数・落札率に与える影響の分析 |
| 道路距離・移動時間 | 直線距離の代わりにGIS・ルーティングAPIを用いた精緻化 |
| 供給制約・担い手不足 | 地域別施工能力制約を導入した実行可能性評価 |
| JV構成・技術者配置 | 建設生産上の制約を加味したロット設計 |

これらを段階的に統合することで、本研究の枠組みはより実務に近い調達政策評価ツールへと発展しうる。
